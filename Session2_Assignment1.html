Unordered List

  • TYpes of List Item Markers

  • Disc-Sets the list item marker to a bullet (default)
  • Circle-Sets the list item marker to a circle
  • Square-Sets the list item marker to a square
  • None-The list items will not be marked

Ordered List

  1. Ordered HTML List - The Type Attribute

  2. Type="1"-The list items will be numbered with numbers (default)
  3. Type="A"-The list items will be numbered with uppercase letters
  4. Type="a"-The list items will be numbered with lowercase letters
  5. Type="I"-The list items will be numbered with uppercase roman numbers
